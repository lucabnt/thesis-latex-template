\textbf{Note:} This is a manual bibliography format. For automatic bibliography management, consider using BibTeX with a .bib file. To use BibTeX:

\begin{enumerate}
    \item Create a References.bib file with your references
    \item Uncomment the bibliography commands in Thesis.tex (lines 249-257)
    \item Comment out the \texttt{\textbackslash include\{Bibliography.tex\}} line
    \item Run: pdflatex, bibtex, pdflatex, pdflatex
\end{enumerate}

\textbf{Example BibTeX entry:}

\begin{verbatim}
@article{author2023,
  author = {Author, A. A. and Author, B. B.},
  title = {Title of the Article},
  journal = {Journal Name},
  year = {2023},
  volume = {10},
  number = {2},
  pages = {123-145}
}
\end{verbatim}

Then cite in text using: \texttt{\textbackslash cite\{author2023\}}
