This folder is for storing the final compiled PDF versions of your thesis.

Recommended naming convention:
- LastName_FirstName_Thesis_YYYY.pdf (main version)
- LastName_FirstName_Thesis_YYYY_Print.pdf (print version if different)
- LastName_FirstName_Thesis_YYYY_University.pdf (if specific to institution)

You may want to keep different versions:
1. Digital submission version (with hyperlinks)
2. Print version (possibly with different margins or formatting)
3. Archive version (PDF/A format for long-term preservation)

Tips:
- Always keep a backup of your final PDF
- Check file size before submission (some institutions have limits)
- Verify that all fonts are embedded
- Test that all hyperlinks work (if applicable)
- Ensure the PDF meets your institution's requirements

To create PDF/A (archival format):
- Use online converters or PDF software
- Or add to your LaTeX preamble:
  \usepackage[a-1b]{pdfx}
